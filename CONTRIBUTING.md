## 🤝 贡献方式

### 方式一：新增问题到知识库

帮助完善问答知识库，让更多人受益。

**步骤：**

1. 打开 `knowledge/faq.jsonl`
2. 添加新问题，格式如下：

```json
{"id": "q025", "question": "你的问题", "answer": "答案内容\n\n详细说明见 docs/xxx.md", "tags": ["标签1", "标签2"], "source": "社区反馈"}
```

3. 更新 `knowledge/topics.json`，将问题 ID 加入对应主题
4. 提交 PR

**注意：**
- 问题 ID 按顺序递增
- 答案要简洁明了
- 标签要与已有标签保持一致

---

### 方式二：补充教程文档

在 `docs/` 目录下添加新教程。

**步骤：**

1. 选择合适的目录：
   - `docs/0-1/` - 入门教程
   - `docs/1-n/` - 进阶教程
   - `docs/plugins/` - 技能相关

2. 创建新的 `.md` 文件

3. 遵循文档格式：
   - 使用「问题导向」标题
   - 每个步骤有 ✅ 目标、📝 操作、⚠️ 常见坑
   - 末尾有「遇到了问题？」指引

4. 提交 PR

---

### 方式三：改进问答工具

优化 `scripts/qa_cli.py` 的搜索算法或功能。

**步骤：**

1. 修改脚本
2. 本地测试
3. 提交 PR

**建议改进方向：**
- 更智能的搜索匹配
- 添加新功能（如语音问答）
- 优化界面显示

---
