## 🛠️ 技能安装方法

```bash
# 从 ClawHub 安装（推荐）
claw install skill-name

# 从 GitHub 安装
claw install https://github.com/user/skill-repo

# 查看已安装技能
claw list

# 更新技能
claw update skill-name
```
