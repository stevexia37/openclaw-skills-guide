## 🎯 OpenClaw 是什么？

**一句话定义：**

> OpenClaw 是一个**本地运行、完全免费、真正能干**的 AI 助手。

想象一下：
- 你有一个私人助理，住在你的电脑里
- 它能帮你处理工作、写代码、管理日程
- 所有数据都在你自己的电脑上，**不上云、不泄露**
- 而且**完全免费**
