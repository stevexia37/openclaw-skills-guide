## 🎯 三大核心功能

### 1️⃣ 文件操作

OpenClaw 能读写、编辑、转换文件：

```
帮我把这个 PDF 转成文字
```

```
帮我整理下载文件夹，按类型分类
```

```
把这个 Excel 表格的数据分析一下
```

### 2️⃣ 定时任务

设置提醒和自动化：

```
每天早上9点提醒我开会
```

```
每周五下午5点提醒我写周报
```

```
10分钟后提醒我休息一下
```

### 3️⃣ 浏览器自动化

操作网页、填表、抓数据：

```
帮我打开这个网站并截图
```

```
去这个网站抓取商品价格
```

## 🛠️ 安装第一个技能

技能让 OpenClaw 变得更强大。

### 从 ClawHub 安装

```
步骤 1: 打开 https://clawhub.ai
步骤 2: 搜索需要的技能
步骤 3: 点击「安装」
步骤 4: 重启 OpenClaw
```

### 推荐新手安装的技能

| 技能 | 功能 | 必要性 |
|------|------|--------|
| **cron** | 定时任务 | ⭐⭐⭐⭐⭐ |
| **pdf** | PDF 处理 | ⭐⭐⭐⭐⭐ |
| **xlsx** | Excel 处理 | ⭐⭐⭐⭐ |
| **browser** | 浏览器操作 | ⭐⭐⭐⭐ |

## 📝 提问技巧

### 好的提问方式 ✅

```
帮我总结这篇文档的三个要点
```

```
把这份 Excel 里销售额前10的产品列出来
```

```
每天早上9点提醒我喝水
```

### 不好的提问方式 ❌

```
帮我弄一下那个东西
```

```
看看这个
```

```
搞一下
```

### 提问公式

> **目标 + 动作 + 细节 = 最佳效果**

```
目标：帮我处理这份报表
动作：找出销售额最高的产品
细节：按月份分类，输出前5名
```
