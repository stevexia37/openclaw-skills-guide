# 第一个技能安装

## 从 ClawHub 安装

1. 打开 ClawHub: https://clawhub.ai
2. 浏览或搜索技能
3. 点击"安装"按钮
4. 在 OpenClaw 中启用技能

## 推荐新手技能

| 技能 | 功能 | 难度 |
|------|------|------|
| cron | 定时任务管理 | ⭐ |
| pdf | PDF 处理 | ⭐ |
| xlsx | Excel 处理 | ⭐ |
| multi-search-engine | 多引擎搜索 | ⭐⭐ |

## 使用技能

安装后，直接在对话中提及技能名称或功能即可：

```
帮我设置一个每天早上9点的提醒
```

OpenClaw 会自动调用 cron 技能来完成任务。
