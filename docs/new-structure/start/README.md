## ⚡ 5 分钟上手

### 1. 打开 OpenClaw

安装后打开，看到配对界面。

### 2. 获取配对码

点击「配对」→ 记下配对码。

### 3. 选择渠道

| 渠道 | 推荐 | 配置难度 |
|------|------|----------|
| **飞书** | ⭐⭐⭐ 企业用户 | ⭐ 简单 |
| Telegram | ⭐⭐ 技术用户 | ⭐⭐ 中等 |
| 钉钉 | ⭐ 国内企业 | ⭐⭐ 中等 |

### 4. 输入配对码

在飞书/Telegram/钉钉中输入配对码。

### 5. 测试消息

发送「你好」测试 AI 响应。
