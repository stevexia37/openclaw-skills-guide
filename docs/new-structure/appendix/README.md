## ❓ 常见问题速查

### 安装问题

| 问题 | 解决方案 |
|------|----------|
| macOS 提示"已损坏" | `sudo xattr -cr /Applications/OpenClaw.app` |
| 配对失败 | 重启 App、检查网络 |

### 渠道问题

| 问题 | 解决方案 |
|------|----------|
| 飞书消息不回复 | 检查权限配置 |
| Telegram Bot 无响应 | 检查 Bot Token |

### 技能问题

| 问题 | 解决方案 |
