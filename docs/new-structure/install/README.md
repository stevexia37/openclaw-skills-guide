# 安装配置

> 5 分钟完成安装配对

---

## 📋 三步安装

### Step 1: 环境准备

| 系统 | 要求 |
|------|------|
| macOS | macOS 12+ (Monterey) |
| Windows | Windows 10/11 |
| Linux | Ubuntu 20.04+ / Debian 11+ |

### Step 2: 下载安装

**下载地址**: [GitHub Releases](https://github.com/openclaw/openclaw/releases)

| 系统 | 文件 |
|------|------|
| macOS | `OpenClaw-x.x.x-macos.dmg` |
| Windows | `OpenClaw-x.x.x-windows.exe` |
| Linux | `curl -fsSL https://get.openclaw.ai | sh` |

### Step 3: 配对连接

1. 打开 OpenClaw
2. 点击「配对」按钮
3. 记下配对码（如 `BHT9-P4UW`）
4. 在渠道输入配对码

---

## ❓ 安装常见问题

| 问题 | 解决方案 |
|------|----------|
| macOS 提示"已损坏" | `sudo xattr -cr /Applications/OpenClaw.app` |
| Windows 无法启动 | 检查防火墙、杀毒软件 |
| 配对失败 | 重启 App、检查网络 |
| 端口被占用 | 检查 18789/18791 端口 |
