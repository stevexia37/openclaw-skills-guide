# OpenClaw 界面展示

## 主界面

OpenClaw 采用简洁的对话界面设计：

```
┌─────────────────────────────────────┐
│  🦞 OpenClaw                    ⚙️  │
├─────────────────────────────────────┤
│                                     │
│  你: 帮我整理下载文件夹              │
│                                     │
│  OpenClaw: 好的，我来帮你整理：      │
│                                     │
│  已整理完成：                        │
│  - 图片 → 图片文件夹 (23个)          │
│  - 文档 → 文档文件夹 (15个)          │
│  - 视频 → 视频文件夹 (8个)           │
│                                     │
│  你: 每天早上9点提醒我开会           │
│                                     │
│  OpenClaw: ✅ 已设置提醒             │
│  - 时间: 每天 09:00                 │
│  - 内容: 开会                       │
│                                     │
├─────────────────────────────────────┤
│  [输入框]                    [发送] │
└─────────────────────────────────────┘
```
