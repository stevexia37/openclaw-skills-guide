# OpenClaw 安装教程

## 系统要求

- macOS 10.15+ / Windows 10+ / Ubuntu 18.04+
- 4GB+ 内存
- 稳定的网络连接

## 安装步骤

### macOS

1. 访问 https://openclaw.ai 下载 `.dmg` 文件
2. 双击打开，拖拽到 Applications
3. 打开应用，允许网络权限
4. 扫码配对

### Windows

1. 访问 https://openclaw.ai 下载 `.exe` 文件
2. 双击运行安装程序
3. 选择安装路径
4. 完成安装后打开应用
5. 扫码配对

### Linux

```bash
# 下载 AppImage
wget https://openclaw.ai/download/linux

# 添加执行权限
chmod +x openclaw.AppImage

# 运行
./openclaw.AppImage
```

## 配对流程

1. 打开应用后显示配对码
2. 使用手机扫描二维码
3. 在手机上确认配对
4. 完成！

## 常见问题

### 无法打开应用

**macOS**: 系统偏好设置 → 安全性与隐私 → 允许打开

**Windows**: 以管理员身份运行

### 配对失败

1. 检查网络连接
2. 重启应用
3. 重新扫描二维码
