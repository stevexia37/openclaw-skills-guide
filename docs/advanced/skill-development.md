# OpenClaw 技能开发指南

## 什么是技能？

技能是 OpenClaw 的扩展模块，让 AI 助手获得新能力。

## 技能结构

```
my-skill/
├── SKILL.md      # 技能文档（必需）
├── skill.py      # 技能脚本
├── templates/    # 模板文件
└── examples/     # 示例文件
```

## SKILL.md 格式

```markdown
# 技能名称

简短描述技能功能。

## 使用方法

描述如何使用这个技能。

## 示例

给出使用示例。

## 注意事项

列出使用限制和注意事项。
```

## 技能类型

1. **文档型技能** - 只有 SKILL.md，提供知识
2. **脚本型技能** - 包含可执行代码
3. **模板型技能** - 提供模板和示例

## 发布技能

1. 创建 GitHub 仓库
2. 添加技能文件
3. 在 ClawHub 提交收录

## 最佳实践

- 保持简洁
- 提供清晰示例
- 处理错误情况
- 保护用户隐私
