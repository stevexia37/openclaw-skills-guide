## 安装 (2分钟)

### macOS

```bash
# 1. 下载 DMG 文件
# 2. 打开 DMG，拖到应用程序
# 3. 如果提示「已损坏」，运行：
xattr -cr /Applications/OpenClaw.app
# 4. 打开应用
```

### Windows

```bash
# 1. 下载 EXE 文件
# 2. 双击安装
# 3. 如果被拦截，点击「更多信息」→「仍要运行」
```

### Linux

```bash
# 下载 AppImage
chmod +x OpenClaw.AppImage
./OpenClaw.AppImage
```

## 配对 (1分钟)

### 方式一：扫码（推荐）

1. 打开 OpenClaw
2. 手机扫码
3. 确认配对

### 方式二：配对码

1. 记下显示的配对码（如 ABCD-1234）
2. 在手机 App 中输入
3. 确认配对

## 开始使用 (2分钟)

### 试试这些

| 说的内容 | 效果 |
|----------|------|
| 你好 | 打招呼 |
| 今天天气 | 查天气 |
| 5分钟后提醒我休息 | 设置提醒 |
| 帮我写一首诗 | 生成内容 |

### 安装技能

```
打开 https://clawhub.ai
搜索需要的技能
点击「安装」
重启 OpenClaw
```

**推荐安装**：cron（定时）、pdf（文档）、xlsx（表格）

## 常见问题

### macOS 提示「已损坏」

```bash
xattr -cr /Applications/OpenClaw.app
```

### Windows 被拦截

点击「更多信息」→「仍要运行」

### 配对失败

1. 检查网络
2. 更新 App
3. 重试配对

### 对话没反应

1. 检查网络
2. 检查登录状态
3. 重启 OpenClaw
