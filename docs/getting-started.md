# OpenClaw 快速入门

## 什么是 OpenClaw？

OpenClaw 是一个本地运行的 AI 助手平台，特点：

- **免费开源** - 完全免费使用
- **本地运行** - 数据在本地，隐私安全
- **多模型支持** - 支持多种 AI 模型，无需翻墙
- **技能系统** - 拥有超过 30,000+ 技能库
- **跨平台** - 支持 macOS, Windows, Linux

## 安装步骤

1. 访问 https://openclaw.ai 下载安装包
2. 双击安装，一路下一步
3. 打开应用，扫码配对
4. 开始使用

## 核心功能

- 自动化日常任务
- 邮件管理
- 文档处理
- 数据分析
- 网页自动化
- 语音转文字
- 图片处理
- 定时任务
