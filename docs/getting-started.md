# OpenClaw 快速入门指南

## 🦞 什么是 OpenClaw？

OpenClaw 是一个**本地运行的 AI 助手平台**。简单说，它就像你的私人助理，住在你的电脑里，帮你处理各种任务。

### 与其他 AI 有什么不同？

| 对比项 | ChatGPT/文心一言 | OpenClaw |
|--------|------------------|----------|
| **数据安全** | 对话存在云端 | **存在你电脑里** |
| **能做什么** | 只能聊天、回答问题 | **能操作你的电脑** |
| **费用** | 每月几百块订阅费 | **完全免费** |
| **隐私** | 数据上传到服务器 | **数据不离开你的电脑** |

### OpenClaw 能做什么？

**日常工作：**
- 📧 自动整理邮件
- 📝 帮你写周报、总结
- 📅 管理日程、设置提醒
- 📊 处理 Excel、PDF 文件

**开发相关：**
- 💻 写代码、调试程序
- 🔄 自动化重复任务
- 🌐 操作浏览器填表、抓数据

**生活相关：**
- 💰 投资理财分析
- 🍳 推荐菜谱、购物清单
- 📚 学习辅导、知识问答

## 🚀 5分钟快速上手

### 第一步：下载安装

1. 打开官网：https://openclaw.ai
2. 点击"下载"，选择你的系统（Mac/Windows/Linux）
3. 下载完成后，双击安装

### 第二步：配对账号

1. 打开 OpenClaw 应用
2. 屏幕上会显示一个**二维码**
3. 用手机扫描二维码
4. 在手机上确认配对
5. 完成！

### 第三步：开始对话

现在你可以和 OpenClaw 对话了！

**试试这些：**
```
帮我总结一下今天的待办事项
```
```
帮我把这个 PDF 转成文字
```
```
每天早上9点提醒我喝水
```

## 🛠️ 安装第一个技能

技能是 OpenClaw 的"超能力"。安装技能后，OpenClaw 就能做更多事情。

### 方法一：从 ClawHub 安装（推荐）

1. 打开 ClawHub：https://clawhub.ai
2. 浏览或搜索你需要的技能
3. 点击"安装"按钮
4. 完成！

### 推荐新手安装的技能

| 技能名 | 能做什么 | 推荐指数 |
|--------|----------|----------|
| **cron** | 定时任务、提醒 | ⭐⭐⭐⭐⭐ |
| **pdf** | PDF 处理、提取文字 | ⭐⭐⭐⭐⭐ |
| **xlsx** | Excel 表格处理 | ⭐⭐⭐⭐ |
| **multi-search-engine** | 多引擎搜索 | ⭐⭐⭐⭐ |

## ❓ 常见问题

### Q: OpenClaw 免费吗？
**A**: 完全免费！OpenClaw 是开源项目，不需要付费。

### Q: 我的对话会被上传吗？
**A**: 不会！所有数据都在你本地电脑上，非常安全。

### Q: 需要翻墙吗？
**A**: 不需要！OpenClaw 支持多种 AI 模型，包括国内模型。

### Q: 支持哪些系统？
**A**: macOS、Windows、Linux 都支持。
