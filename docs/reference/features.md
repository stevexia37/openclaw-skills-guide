# OpenClaw 功能大全

## 🎯 核心功能一览

### 📧 邮件管理

**能做什么：**
- 自动整理收件箱
- 筛选重要邮件
- 自动回复简单邮件
- 发送定时邮件

**示例对话：**
```
帮我看看今天的重要邮件有哪些
```

---

### 📝 文档处理

**能做什么：**
- PDF 提取文字
- Word 文档编辑
- Excel 表格处理
- PPT 生成

---

### 🌐 浏览器自动化

**能做什么：**
- 自动填表
- 网页数据抓取
- 自动登录网站
- 定时检查网页更新

---

### ⏰ 定时任务

**能做什么：**
- 设置提醒
- 定时执行任务
- 周期性工作自动化

---

### 💻 代码助手

**能做什么：**
- 写代码、调试程序
- 代码审查
- 解释代码

---

## 🛠️ 技能系统

技能是 OpenClaw 的扩展模块。安装不同技能，OpenClaw 就能获得新能力。

### 安装技能

1. 访问 https://clawhub.ai
2. 搜索或浏览技能
3. 点击安装
