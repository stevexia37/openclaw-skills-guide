# 附录 D：最佳实践

> 💡 高效使用 OpenClaw 的技巧和建议

---

## 🎯 提问最佳实践

### ✅ 好的提问方式

**明确目标：**
```
帮我分析这个 Excel 表格，找出销售额前10的产品
```

**提供细节：**
```
把这份 PDF 转成文字，只提取第1-5页
```

**使用公式：**
> **目标 + 动作 + 细节 = 最佳效果**

### ❌ 避免的提问方式

```
帮我弄一下那个东西  ← 不明确
看看这个            ← 没说要做什么
搞一下              ← 没有目标
```

---

## 📁 文件操作最佳实践

### 操作前先确认

```
你：我要删除这个文件夹，先确认一下内容
OpenClaw：这个文件夹有 15 个文件，确认删除吗？
你：确认
```

### 重要文件先备份

```
你：帮我备份这个文件再修改
OpenClaw：已创建备份：report_backup.md
```

### 使用版本控制

```
你：帮我保存修改版本，标记"v2"
OpenClaw：已保存为 report_v2.md
```

---

## ⏰ 定时任务最佳实践

### 提醒内容要具体

```
❌ 每天提醒我
✅ 每天早上9点提醒我开会
```

### 设置合理间隔

```
❌ 每分钟提醒我
✅ 每小时提醒我休息
```

### 定期检查任务

```
你：列出我的所有定时任务
OpenClaw：（列出任务列表）
你：删除不需要的任务
```

## 🔒 安全最佳实践

### 不要泄露敏感信息

```
❌ 我的密码是123456
❌ 我的银行卡号是...
✅ 这些信息不要告诉 AI
```

### 重要操作先确认

```
你：删除所有文件
OpenClaw：⚠️ 这是一个危险操作，确认吗？
你：取消
```

### 保持数据本地

```
确保 OpenClaw 配置为本地模式
不要上传数据到云端
```

### 建立常用模板

## 🔄 工作流程最佳实践

### 每日流程

```
早上：
├── OpenClaw 提醒今日日程
├── 快速查看重要邮件
└── 准备今日待办

工作中：
├── 用 OpenClaw 处理文件
├── 定时提醒休息
└── 自动化重复任务

晚上：
├── OpenClaw 生成日报
├── 整理今日文件
└── 规划明日任务
```

### 每周流程

```
周一：规划本周任务
周五：生成周报总结
周末：整理和复盘
```

## 💡 进阶技巧

### 上下文记忆

```
OpenClaw 会记住对话内容
你可以：
- 在之前的基础上继续
- 不需要重复说明
- 用"继续"、"加上"等词
```

### 技能组合

```
多个技能可以配合使用：
"搜索最近新闻，然后总结成报告"
（搜索 + 总结技能）
```

### 自定义配置

```
根据你的需求调整：
- 默认文件路径
- 提醒格式
- 输出风格
```
