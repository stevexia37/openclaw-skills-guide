## 🚀 如何提交技能？

想把你的技能收录到 awesome-openclaw-skills？

### 提交要求

1. 技能已发布到 ClawHub
2. 技能已发布到 GitHub（github.com/openclaw/skills）
3. 技能质量达标（无垃圾、无重复、有描述）

### 提交方法

1. Fork VoltAgent/awesome-openclaw-skills
2. 在 README.md 中添加技能信息
3. 提交 PR

**格式**：
```
- [技能名称](GitHub链接) - 功能描述。([ClawHub](ClawHub链接))
```
