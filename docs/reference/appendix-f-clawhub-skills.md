## 📦 安装方法

### ClawHub 安装

访问：https://clawhub.ai

```
步骤 1: 打开 ClawHub
步骤 2: 搜索技能名称
步骤 3: 点击「安装」按钮
步骤 4: 重启 OpenClaw
```

### 命令行安装

```bash
# 安装技能
openclaw skill install <技能名>

# 示例
openclaw skill install pdf
```
