# 附录 E：官方自带技能

> OpenClaw 安装后自带的基础技能

---

## 📋 核心技能列表

以下是 OpenClaw 官方内置的技能，无需额外安装：

### 📁 文件操作

| 技能 | 功能 | 使用示例 |
|------|------|----------|
| **file_reader** | 读取文件内容 | 「帮我读取这个文件」 |
| **file_writer** | 写入文件内容 | 「帮我写一个文件」 |
| **file_manager** | 文件管理操作 | 「列出当前目录文件」 |

### 🌐 浏览器操作

| 技能 | 功能 | 使用示例 |
|------|------|----------|
| **browser_use** | 浏览器自动化 | 「打开这个网站并截图」 |
| **browser_visible** | 可视化浏览器 | 「帮我看看这个网页」 |

### 📄 文档处理

| 技能 | 功能 | 使用示例 |
|------|------|----------|
| **pdf** | PDF 处理 | 「把这个 PDF 转成文字」 |
| **xlsx** | Excel 处理 | 「分析这个 Excel 表格」 |
| **docx** | Word 处理 | 「编辑这个 Word 文档」 |
| **pptx** | PPT 处理 | 「创建一个演示文稿」 |

### ⏰ 定时任务

| 技能 | 功能 | 使用示例 |
|------|------|----------|
| **cron** | 定时任务管理 | 「每天9点提醒我开会」 |
| **scheduler** | 任务调度 | 「每周五提醒我写周报」 |

### 💻 开发工具

| 技能 | 功能 | 使用示例 |
|------|------|----------|
| **terminal** | 命令行执行 | 「运行这个命令」 |
| **code_helper** | 代码辅助 | 「帮我写一段代码」 |

---

## 🔧 技能使用方法

### 直接对话使用

安装后直接在对话中使用：

```
你: 帮我把这个 PDF 转成文字
OpenClaw: （自动调用 pdf 技能处理）
```

### 查看已安装技能

```
你: 列出我安装的技能
OpenClaw: 你已安装以下技能：
1. pdf - PDF处理
2. xlsx - Excel处理
3. cron - 定时任务
...
```
