# OpenClaw 使用技巧

## 💡 提问技巧

### 如何更好地与 OpenClaw 沟通？

**✅ 好的提问方式：**
```
帮我总结这篇文档的三个要点
把这份 Excel 表格里销售额前10的产品列出来
每天早上9点提醒我喝水
```

**❌ 不好的提问方式：**
```
帮我弄一下那个东西
看看这个
搞一下
```

### 提问公式

**目标 + 动作 + 细节 = 最佳效果**

```
目标：帮我处理这份报表
动作：找出销售额最高的产品
细节：按月份分类，输出前5名
```

## 🔧 常用命令速查

### 文件操作

| 你说的话 | OpenClaw 会做什么 |
|----------|-------------------|
| "帮我读取这个文件" | 读取并显示文件内容 |
| "把这个文件转成PDF" | 转换文件格式 |
| "帮我整理这个文件夹" | 按类型分类文件 |

### 定时任务

| 你说的话 | OpenClaw 会做什么 |
|----------|-------------------|
| "每天早上9点提醒我开会" | 设置每日提醒 |
| "每周五提醒我写周报" | 设置每周提醒 |
| "取消所有提醒" | 清除定时任务 |

### 数据处理

| 你说的话 | OpenClaw 会做什么 |
|----------|-------------------|
| "帮我分析这个Excel" | 读取并分析表格数据 |
| "把这些数据画成图表" | 生成可视化图表 |
| "找出这份数据的异常值" | 数据异常检测 |

## 🎯 场景示例

### 场景1：每天的工作汇报

**你说：**
```
帮我根据今天的日程，写一份简短的工作汇报
```

**OpenClaw 会：**
1. 读取你的日程安排
2. 整理完成的任务
3. 生成汇报文档

### 场景2：学习新知识

**你说：**
```
我想学习 Python 基础，帮我制定一个学习计划
```

**OpenClaw 会：**
1. 了解你的背景
2. 制定个性化学习计划
3. 提供学习资源推荐

### 场景3：投资理财

**你说：**
```
帮我查一下比特币现在的价格，分析一下趋势
```

**OpenClaw 会：**
1. 获取实时价格数据
2. 分析价格走势
3. 给出简单分析报告

## 🚀 进阶技巧

### 1. 上下文记忆

OpenClaw 会记住对话内容，你可以：

```
你：帮我写一个 Python 爬虫
OpenClaw：（写好代码）

你：加一个异常处理
OpenClaw：（在之前的基础上添加）
```

### 2. 技能组合

多个技能可以配合使用：

```
帮我搜索最近的 AI 新闻，然后总结成一份报告
```

这会用到：
- 搜索技能（获取新闻）
- 总结技能（生成报告）

### 3. 文件引用

直接说文件名，OpenClaw 能找到：

```
帮我看看 report.docx 里写了什么
```

## ⚠️ 注意事项

1. **敏感信息不要说**：密码、银行卡号等
2. **重要文件先备份**：修改前让 OpenClaw 先备份
3. **确认再执行**：重要操作先确认再让 OpenClaw 执行
