# 如何配对账号并开始使用？

> 🎯 本章目标：完成账号配对，开始第一次对话

---

## ✅ 目标

完成本章后，你将：
- 配对 OpenClaw 账号
- 开始第一次对话
- 知道如何提问

---

## 📝 操作：配对账号

### 方式一：二维码配对（推荐）

1. 打开 OpenClaw 应用
2. 屏幕显示**二维码**
3. 用手机扫描二维码
4. 在手机上确认配对
5. 完成！

### 方式二：配对码配对

如果没有摄像头：

1. 记下屏幕显示的**配对码**（如：`ABCD-1234`）
2. 在手机 OpenClaw App 中输入配对码
3. 确认配对
4. 完成！

---

## ⚠️ 常见坑：配对失败

### 问题：二维码扫不出来

**原因：**
- 屏幕亮度低
- 二维码过期

**解决：**
1. 调亮屏幕亮度
2. 重新打开应用生成新二维码

---

### 问题：配对码输入无效

**原因：**
- 配对码过期（通常 5 分钟）
- 输入错误

**解决：**
1. 重新获取配对码
2. 仔细检查大小写和数字

---

### 问题：配对后无反应

**原因：**
- 网络延迟
- 服务器繁忙

**解决：**
1. 等待 1-2 分钟
2. 重启 OpenClaw 应用

---

## 📝 操作：第一次对话

配对成功后，试试这些对话：

### 认识 OpenClaw

```
你好，介绍一下你自己
```

### 基本功能测试

```
帮我列出当前目录的文件
```

```
今天是什么日期？
```

---

## 📝 操作：学习提问技巧

### ✅ 好的提问方式

**明确目标：**
```
帮我分析这个 Excel 表格，找出销售额前10的产品
```

**提供细节：**
```
把这份 PDF 转成文字，只提取第1-5页
```

**公式：目标 + 动作 + 细节**

---

### ❌ 避免的提问方式

```
帮我弄一下那个东西  ← 不明确
看看这个            ← 没说要做什么
搞一下              ← 没有目标
```

---

## 📝 操作：安装第一个技能

技能让 OpenClaw 更强大。

### 推荐：安装定时提醒技能

1. 在对话中说：
```
帮我安装 cron 技能
```

2. 或打开 https://clawhub.ai 搜索 `cron`

3. 点击「安装」

### 测试技能

安装后试试：
```
每天早上9点提醒我开会
```

---

## ⚠️ 常见坑：技能安装失败

### 问题：找不到技能

**解决：**
1. 确认技能名称正确
2. 去 ClawHub 搜索确认存在

---

### 问题：安装后不生效

**解决：**
1. 重启 OpenClaw
2. 检查技能目录
