## 第二步：试试对话

### 测试对话功能

输入以下内容测试：

| 试试说 | 预期效果 |
|--------|----------|
| 「你好」 | 打招呼，介绍自己 |
| 「现在几点了」 | 告诉当前时间 |
| 「今天天气」 | 查询天气信息 |
| 「帮我写一首诗」 | 生成一首诗 |
| 「讲个笑话」 | 讲一个笑话 |

### 如果对话不响应

1. 检查网络连接
2. 检查模型是否配置
3. 查看错误日志
4. 运行 `openclaw doctor`

## 第四步：安装技能

### 推荐安装这些技能

| 技能 | 功能 | 安装理由 |
|------|------|----------|
| **cron** | 定时提醒 | 必装！设置提醒任务 |
| **pdf** | 处理 PDF | 读取、提取、合并 PDF |
| **xlsx** | 处理表格 | 分析 Excel 数据 |
| **docx** | 处理 Word | 读取、修改 Word 文档 |

### 安装方法

**方法一：ClawHub**
1. 打开 https://clawhub.ai
2. 搜索技能名称
3. 点击「安装」

**方法二：命令行**
```
安装技能 pdf
```

**安装后重启 OpenClaw 生效**

## 常见问题

<details>
<summary><b>Q: 对话没反应？</b></summary>

**检查项**：
1. 网络是否连接
2. 账号是否登录
3. 是否有错误提示

**解决方法**：
- 重启 OpenClaw
- 检查日志：`openclaw logs`
- 运行诊断：`openclaw doctor`
</details>

<details>
<summary><b>Q: 技能安装失败？</b></summary>

**可能原因**：
1. 网络问题
2. 技能不存在
3. 权限问题

**解决方法**：
- 检查网络
- 确认技能名称正确
- 尝试手动安装
</details>

<details>
<summary><b>Q: 提醒不生效？</b></summary>

**检查项**：
1. OpenClaw 是否在运行
2. 通知权限是否开启
3. 时间格式是否正确

**解决方法**：
- 确保 OpenClaw 在后台运行
- 系统设置中开启通知权限
- 使用正确格式重新设置
</details>
