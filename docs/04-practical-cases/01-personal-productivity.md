### 使用前

| 任务 | 耗时 |
|------|------|
| 整理邮件 | 30 分钟 |
| 处理表格 | 40 分钟 |
| 写周报 | 20 分钟 |

| 任务 | 耗时 | OpenClaw 帮什么 |
|------|------|-----------------|

| **总计** | **20 分钟** | ⬇️ **节省 70 分钟** |

### 具体对话示例

```
你：帮我看看今天的重要邮件
OpenClaw：找到 3 封重要邮件：
1. 来自老板的会议通知

3. 来自财务的报销确认

你：帮我回复第2封，说已收到，稍后处理
OpenClaw：已发送回复
```

```
你：帮我分析这个 Excel，找出销售额前10的产品
OpenClaw：分析完成，前10产品是：
1. 产品A - ¥120,000
2. 产品B - ¥98,000
...
```

```
你：帮我根据今天的日程写周报
OpenClaw：周报已生成：
本周完成：
- 参加了3次会议
- 处理了15封邮件
- 完成了数据报表
...
```
