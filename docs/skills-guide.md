# 技能使用指南

## 🤔 什么是技能？

技能就像手机的 App，给 OpenClaw 安装技能后，它就能做更多事情。

## 📦 技能分类

### 办公效率类

| 技能名 | 功能 | 适合人群 |
|--------|------|----------|
| **pdf** | PDF 处理、提取文字 | 所有人 |
| **xlsx** | Excel 表格处理 | 办公族 |
| **docx** | Word 文档处理 | 办公族 |
| **pptx** | PPT 生成和编辑 | 演讲者 |

### 自动化类

| 技能名 | 功能 | 适合人群 |
|--------|------|----------|
| **cron** | 定时任务管理 | 所有人 |
| **browser** | 浏览器自动化 | 开发者 |
| **file-watcher** | 文件监控 | 开发者 |

### 开发类

| 技能名 | 功能 | 适合人群 |
|--------|------|----------|
| **code-helper** | 代码辅助 | 开发者 |
| **git** | Git 操作 | 开发者 |
| **terminal** | 命令行助手 | 开发者 |

### 生活类

| 技能名 | 功能 | 适合人群 |
|--------|------|----------|
| **weather** | 天气查询 | 所有人 |
| **recipe** | 菜谱推荐 | 家务族 |
| **news** | 新闻聚合 | 所有人 |

## 🚀 如何安装技能

### 方法一：从 ClawHub 安装（推荐新手）

1. 打开 https://clawhub.ai
2. 搜索你想要的技能
3. 点击"安装"按钮
4. 等待安装完成

### 方法二：手动安装

1. 下载技能文件夹
2. 放到 OpenClaw 技能目录
3. 重启 OpenClaw

---

## 💡 技能使用技巧

### 不需要记命令

安装技能后，直接用自然语言描述需求：

```
安装了 pdf 技能后，你可以说：
"帮我把这个 PDF 转成文字"

安装了 xlsx 技能后，你可以说：
"帮我分析这个 Excel 表格"
```

### 技能会自动激活

OpenClaw 会根据你的需求自动调用对应技能：

```
你：每天早上提醒我喝水
OpenClaw：（自动使用 cron 技能）
```

## 🔧 常见问题

### Q: 技能安装失败怎么办？
**A**: 检查网络连接，或尝试重新安装

### Q: 技能不生效怎么办？
**A**: 重启 OpenClaw，或检查技能是否正确安装

### Q: 如何卸载技能？
**A**: 在设置中找到技能管理，点击卸载
