# 什么是 OpenClaw？

> 用最简单的话，讲清楚 OpenClaw 是什么

---

## 🤔 一句话解释

**OpenClaw 就像一个住在电脑里的私人助理**，能帮你干活、聊天、处理各种任务。

---

## 🆚 和 ChatGPT 有什么区别？

### ChatGPT

```
你：帮我整理这个文件夹
ChatGPT：抱歉，我无法访问您的电脑文件...
```

**只能聊天，不能干活**

### OpenClaw

```
你：帮我整理这个文件夹
OpenClaw：好的，我来帮你按类型分类...
（实际操作你的电脑，整理完成）
```

**既能聊天，又能干活**

---

## 🎯 形象比喻

| 工具 | 比喻 |
|------|------|
| ChatGPT | 像一个**电话客服**，只能说话，不能帮你做事 |
| OpenClaw | 像一个**住在你家的助理**，能说话，也能帮你干活 |

---

## ✨ OpenClaw 能做什么？

### 📁 文件处理

```
帮我把这个 PDF 转成文字
帮我把这些图片整理到一个文件夹
帮我把这个 Excel 表格分析一下
```

### 📅 日程管理

```
每天早上9点提醒我开会
每周五提醒我写周报
帮我看看今天有什么待办事项
```

### 💻 编程开发

```
帮我写一个 Python 爬虫
这段代码有什么问题？
帮我优化这个函数
```

### 🌐 浏览器操作

```
帮我去这个网站抓取数据
帮我自动填表
每天帮我查一下比特币价格
```

### 📧 邮件管理

```
帮我看看今天的重要邮件
帮我回复这封邮件
帮我整理收件箱
```

## 🔒 安全吗？

### 非常安全！

| 问题 | 答案 |
|------|------|
| 对话会被上传吗？ | ❌ 不会！所有数据在你电脑里 |
| 会泄露隐私吗？ | ❌ 不会！数据不上云 |
| 需要网络吗？ | 部分功能需要，但核心功能可离线 |
| 会被别人看到吗？ | ❌ 不会！只有你能看到 |

## 🎓 需要什么基础？

### 完全不需要编程基础！

OpenClaw 是**自然语言交互**：

```
你只需要说人话：
"帮我整理这个文件夹"
"每天提醒我喝水"
"帮我分析这个表格"
```

就像和一个真人说话一样简单！
