# 🦞 OpenClaw 技能百科 — 快速上手指南

> 5 分钟，从零基础到第一个技能运行成功。

## 前置要求

- macOS 10.15+ / Windows 10+ / Ubuntu 18.04+
- 终端（Terminal / PowerShell / Bash）
- 网络连接（首次安装需要）

## 三步上手

### 1️⃣ 安装 OpenClaw

```bash
# 一键安装脚本
curl -fsSL https://raw.githubusercontent.com/openclaw/openclaw/main/setup.sh | bash
```

### 2️⃣ 初始化

```bash
openclaw init
```

### 3️⃣ 运行第一个技能

```bash
openclaw skill run hello
```
