## 🟢 阶段一：入门 (30分钟)

> **目标：成功安装并完成配对，能够进行基本对话**

### 第1步：环境准备 (5分钟)

**检查系统要求：**

| 系统 | 最低要求 | 推荐配置 |
|------|----------|----------|
| macOS | 10.15 Catalina | 12.0+ M1/M2 |
| Windows | Windows 10 | Windows 11 |
| Linux | Ubuntu 18.04 | Ubuntu 22.04 |

**检查方法：**
- macOS：点击左上角苹果图标 → 关于本机
- Windows：右键此电脑 → 属性
- Linux：终端输入 `lsb_release -a`

**需要准备：**
- ✅ 网络连接
- ✅ 2GB 磁盘空间
- ✅ 8GB 内存（推荐）

👉 [详细教程：环境准备](./docs/0-1/01-env-prepare.md)

---

### 第2步：下载安装 (10分钟)

**下载地址：**
- 官网：https://openclaw.ai
- GitHub：https://github.com/openclaw

**安装方法：**

| 系统 | 方法 |
|------|------|
| macOS | 打开 DMG → 拖到应用程序 |
| Windows | 运行 EXE → 按提示安装 |
| Linux | 运行 AppImage 或解压 tar.gz |

**常见问题：**

<details>
<summary>macOS 提示「已损坏」？</summary>

打开终端执行：
```bash
xattr -cr /Applications/OpenClaw.app
```
然后重新打开应用。
</details>

<details>
<summary>Windows 被杀毒拦截？</summary>

点击「更多信息」→「仍要运行」

或暂时关闭杀毒软件。
</details>

👉 [详细教程：安装指南](./docs/0-1/02-install.md)

---

### 第3步：配对账号 (10分钟)

**两种配对方式：**

**方式一：扫码配对（推荐）**
1. 打开 OpenClaw
2. 手机扫码
3. 确认配对

**方式二：配对码**
1. 记下显示的配对码（如 ABCD-1234）
2. 在手机 App 中输入配对码
3. 确认配对

**配对成功标志：**
- 显示「已连接」
- 能看到账号信息

👉 [详细教程：配对账号](./docs/0-1/03-config.md)

---

### 第4步：首次运行 (5分钟)

**试试这些命令：**

| 试试说 | 效果 |
|--------|------|
| 「你好」 | 打招呼 |
| 「今天天气」 | 查天气 |
| 「帮我写一个待办清单」 | 创建清单 |
| 「5分钟后提醒我休息」 | 设置提醒 |

**恭喜！你已经完成入门！** 🎉

## 🟡 阶段二：进阶 (45分钟)

> **目标：掌握安全设置、数据备份、技能安装**

### 第1步：安全设置 (10分钟)

**隐私保护：**

| 设置项 | 建议 |
|--------|------|
| 数据存储 | 默认本地 |
| 对话保存 | 可选关闭 |
| 敏感信息 | 不透露密码等 |

**推荐设置：**
1. 打开设置
2. 隐私与安全
3. 根据需要调整

👉 [详细教程：安全设置](./docs/1-n/01-security.md)

---

### 第2步：备份数据 (8分钟)

**数据位置：**
- macOS/Linux：`~/.copaw/`
- Windows：`C:\Users\用户\.copaw\`

**备份方法：**
1. 复制整个 `.copaw` 目录
2. 保存到安全位置（云盘/外部硬盘）

**建议频率：**
- 每周备份一次
- 重要操作前备份

👉 [详细教程：备份数据](./docs/1-n/02-backup.md)

---

### 第3步：性能监控 (10分钟)

**查看状态：**
```bash
openclaw doctor
```

**常见问题：**

| 问题 | 解决方法 |
|------|----------|
| 运行慢 | 切换更快的模型 |
| 内存高 | 关闭不用的技能 |
| 响应慢 | 检查网络 |

👉 [详细教程：性能监控](./docs/1-n/03-monitor.md)

---

### 第4步：安装技能 (15分钟)

**什么是技能？**
技能是 OpenClaw 的「超能力」扩展。

**安装方法：**
1. 打开 https://clawhub.ai
2. 搜索技能
3. 点击「安装」
4. 重启 OpenClaw

**新手推荐技能：**

| 技能 | 功能 | 必装指数 |
|------|------|----------|
| cron | 定时提醒 | ⭐⭐⭐⭐⭐ |
| pdf | 处理 PDF | ⭐⭐⭐⭐ |
| xlsx | 处理表格 | ⭐⭐⭐⭐ |
| docx | 处理 Word | ⭐⭐⭐ |

👉 [详细教程：扩展技能](./docs/1-n/04-scale.md)
