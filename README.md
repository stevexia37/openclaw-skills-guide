## 📖 这是什么？

**OpenClaw** 是一个**本地运行的 AI 助手平台**。

想象一下：
- 你有一个私人助理，住在你的电脑里
- 它能帮你处理工作、写代码、管理日程
- 所有数据都在你自己的电脑上，**不上云、不泄露**
- 而且**完全免费**

## 🚀 5分钟上手

### 第一步：下载安装

| 系统 | 下载方式 |
|------|----------|
| macOS | [下载 .dmg](https://openclaw.ai) → 双击安装 |
| Windows | [下载 .exe](https://openclaw.ai) → 双击安装 |
| Linux | [下载 AppImage](https://openclaw.ai) → `chmod +x && ./OpenClaw.AppImage` |

👉 [详细安装教程](./docs/installation.md)（含常见报错解决方案）

### 第二步：配对账号

1. 打开 OpenClaw
2. 扫描屏幕上的二维码
3. 手机确认配对
4. 完成！

### 第三步：开始使用

```
帮我总结这份文档
每天早上9点提醒我开会
帮我把这个 PDF 转成文字
```

```

**日程管理**
```
