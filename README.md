## 🚀 5分钟上手

### 第一步：安装

| 系统 | 方法 |
|------|------|
| macOS | 官网下载 .dmg → 双击安装 |
| Windows | 官网下载 .exe → 双击安装 |
| Linux | AppImage → `chmod +x && ./` |

👉 [详细安装教程](./docs/01-getting-started/02-installation.md)（含报错解决）

### 第二步：配对

打开 OpenClaw → 扫二维码 → 完成！

### 第三步：对话

```
你: 帮我整理下载文件夹
OpenClaw: 已整理完成！
  - 图片 → 图片文件夹 (23个)
  - 文档 → 文档文件夹 (15个)
```

## 🎯 能帮你做什么？

### 💼 工作场景

- 文档处理：PDF转文字、Excel分析、Word编辑
- 日程管理：提醒事项、待办整理、周报生成
- 邮件处理：筛选重要邮件、自动回复

### 💻 开发场景

- 代码助手：写代码、调试、解释代码
- 自动化：定时脚本、批量处理
- 工具类：JSON格式化、代码片段

### 🏠 生活场景

- 学习辅助：翻译文档、解释概念、制定计划
- 投资理财：价格监控、数据分析
- 家务管理：提醒事项、购物清单
