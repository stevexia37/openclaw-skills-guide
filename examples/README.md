## 🎯 如何使用示例

### 学习技能编写

```bash
# 查看示例技能结构
ls examples/xiaohongshu-writer/

# 阅读技能说明
cat examples/xiaohongshu-writer/README.md
```

### 安装完整技能

```bash
# 从 ClawHub 安装（推荐）
claw install skill-name

# 或从 GitHub 安装
claw install https://github.com/user/skill-repo
```
